**附注**：

- 这个表主要作用是专家级性能监视，或者当为MySQL开发性能相关的扩展。 
- 由于`INFORMATION_SCHEMA` 是一种监控MySQL服务器的通用方式，用这些表而不是对应的InnoDB系统表为您开发任何新的监测应用。
- 你必须有`PROCESS`权限才能查询这个表。
