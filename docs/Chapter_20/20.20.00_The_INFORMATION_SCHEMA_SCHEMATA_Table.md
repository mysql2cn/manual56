下面的语句是等价的：

	SELECT SCHEMA_NAME AS `Database`
	  FROM INFORMATION_SCHEMA.SCHEMATA
	  [WHERE SCHEMA_NAME LIKE 'wild']
	
	SHOW DATABASES
	  [LIKE 'wild']
