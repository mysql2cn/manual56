本小节描述了[BEGIN ... END]()复合语句及其他适用于存储程序中的语句语法，如存储过程、函数、触发器和事件，这些对象的SQL代码被存储在MSYQL服务器端并且之后可以对它们进行调用。（详见：[19, Stored Programs and Views]())。

一个复合语句就是一个包含其他”块“的“块”，其中“块包括变量的声明、异常的处理、游标以及流程控制结构和逻辑判断。
