13.3.7. XA 事务

mysql可以使用 SET autocommit, START TRANSACTION, COMMIT, 和 ROLLBACK　（详见　13.3.1, “START TRANSACTION, COMMIT　和 ROLLBACK 语法”）语句来支持局部事务。此外，mysql 对 XA 事务的支持使得mysql在分布式事务当中可以进行分段提交。（详见 13.3.7, “XA 事务”）
